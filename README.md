# YourMate

A personal coaching Manager Assistant designed to streamline the administration of coaching centers and tutoring businesses.

## Features

- **Student Registration**: Easily register and manage student profiles with personal details, contact information, and enrollment history.
- **Attendance Management**: Track daily attendance, generate attendance reports, and monitor student participation patterns.
- **Fees Management**: Handle fee collection, track payments, manage dues, and generate financial reports.
- **Reminder and Notes System**: Set reminders for important events, exams, and meetings; maintain notes for individual students or batches.
- **Homework Assignment**: Assign, track, and review homework tasks for students with deadline management.
- **Report and Analysis of Marks**: Record exam scores, generate performance reports, and analyze student progress over time with visual insights.
